# Источники данных

В проектах используются открытые датасеты Kaggle. Сырые CSV-файлы не хранятся в репозитории, чтобы не перегружать GitHub.

- Brazilian E-Commerce Public Dataset by Olist
- Credit Card Fraud Detection
- Store Sales - Time Series Forecasting
- Telco Customer Churn
- MovieLens
- Online Retail / Customer Segmentation

CSV-файлы нужно скачать с Kaggle и положить в `data/raw` соответствующего проекта.
