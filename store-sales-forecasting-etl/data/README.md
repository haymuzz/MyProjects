# Данные проекта

В проекте используется датасет **Store Sales - Time Series Forecasting** с Kaggle.

Ссылка на датасет: https://www.kaggle.com/competitions/store-sales-time-series-forecasting

Файлы данных не добавлены в репозиторий, так как датасеты Kaggle могут быть крупными и требуют отдельного скачивания.

## Как скачать

```bash
kaggle competitions download -c store-sales-time-series-forecasting
unzip *.zip -d data/raw
```

После скачивания файлы нужно положить в папку:

```text
store-sales-forecasting-etl/data/raw/
```
