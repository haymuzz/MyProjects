# Python Practice

Папка с небольшими практическими задачами по Python, pandas, продуктовой аналитике и подготовке данных. Каждый файл можно запускать отдельно: внутри есть небольшой пример данных и готовое решение.

| № | Файл | Что отрабатывается |
|---|------|--------------------|
| 1 | `01_clean_sales_data.py` | Очистка данных, типы, пропуски, дубликаты, расчет выручки |
| 2 | `02_rfm_segmentation.py` | RFM-анализ и сегментация клиентов |
| 3 | `03_ab_test_checker.py` | Проверка A/B-теста по конверсии |
| 4 | `04_cohort_retention.py` | Когортный анализ и retention matrix |
| 5 | `05_outlier_detection.py` | Поиск выбросов методом IQR |
| 6 | `06_sql_like_window_metrics.py` | Оконные метрики в pandas в стиле SQL |
| 7 | `07_api_json_normalization.py` | Нормализация вложенного JSON из API |
| 8 | `08_feature_engineering_churn.py` | Feature engineering для модели оттока |
| 9 | `09_sales_forecast_baseline.py` | Бейзлайн-прогноз продаж по скользящему среднему |
| 10 | `10_data_quality_report.py` | Отчет о качестве данных |

## Запуск

```bash
python 01_clean_sales_data.py
```
