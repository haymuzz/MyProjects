# Python Practice

10 самостоятельных практических задач по аналитике данных. Каждый файл запускается отдельно и содержит полный цикл: подготовка данных, расчёт метрик, визуализация и сохранение выводов.

## Список задач

1. `01_sales_cohort_analysis.py` — когортный анализ продаж.
2. `02_ab_test_analysis.py` — проверка A/B-теста.
3. `03_churn_features.py` — признаки для анализа оттока.
4. `04_rfm_segmentation.py` — RFM-сегментация клиентов.
5. `05_forecast_baseline.py` — baseline-прогноз продаж.
6. `06_data_quality_checks.py` — отчёт по качеству данных.
7. `07_api_log_analysis.py` — анализ логов API.
8. `08_product_funnel.py` — продуктовая воронка.
9. `09_unit_economics.py` — юнит-экономика.
10. `10_text_feedback_analysis.py` — анализ текстовых отзывов.

## Запуск

```bash
python 01_sales_cohort_analysis.py
```
