# SQL Practice

В папке 10 законченных SQL-задач в стиле аналитического портфолио.

Каждый файл содержит:

- демонстрационные данные через CTE;
- расчёт бизнес-метрик;
- оконные функции;
- итоговый `SELECT`;
- текстовый аналитический вывод в результате запроса.

Файлы можно запускать в PostgreSQL-подобной среде.
