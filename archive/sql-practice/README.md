# SQL Practice

10 SQL-задач для демонстрации аналитического мышления и владения PostgreSQL-синтаксисом.

## Используется

- CTE;
- агрегирующие функции;
- оконные функции;
- `CASE WHEN`;
- расчёт бизнес-метрик;
- ранжирование и накопительные суммы.

## Список задач

1. `01_revenue_by_month.sql` — динамика выручки по месяцам.
2. `02_top_customers_by_revenue.sql` — топ клиентов по выручке.
3. `03_repeat_purchase_rate.sql` — доля повторных покупок.
4. `04_rfm_segmentation.sql` — RFM-сегментация.
5. `05_sales_forecast_features.sql` — признаки для прогноза продаж.
6. `06_churn_rate_by_segment.sql` — отток по сегментам.
7. `07_window_order_metrics.sql` — оконные метрики заказов.
8. `08_ab_test_conversion.sql` — конверсия A/B-теста.
9. `09_delivery_sla_analysis.sql` — анализ SLA доставки.
10. `10_data_quality_checks.sql` — проверки качества данных.
