# SQL Practice

Папка с практическими SQL-задачами для аналитика данных. Запросы написаны в стиле PostgreSQL и покрывают агрегаты, JOIN, CTE, оконные функции, продуктовые метрики и проверки качества данных.

| № | Файл | Что отрабатывается |
|---|------|--------------------|
| 1 | `01_revenue_by_month.sql` | Месячная выручка, количество заказов, средний чек |
| 2 | `02_top_customers_by_revenue.sql` | Топ клиентов по выручке |
| 3 | `03_repeat_purchase_rate.sql` | Доля клиентов с повторной покупкой |
| 4 | `04_cohort_retention.sql` | Когортное удержание клиентов |
| 5 | `05_rfm_segmentation.sql` | RFM-сегментация на SQL |
| 6 | `06_churn_rate_by_segment.sql` | Churn rate по сегментам и тарифам |
| 7 | `07_window_order_metrics.sql` | Оконные функции по заказам клиента |
| 8 | `08_ab_test_conversion.sql` | Подготовка метрик A/B-теста |
| 9 | `09_delivery_sla_analysis.sql` | Анализ SLA доставки |
| 10 | `10_data_quality_checks.sql` | Проверки качества данных |

## Примечание

Названия таблиц и полей можно адаптировать под конкретный датасет проекта. Основная цель папки — показать владение аналитическим SQL и умение формировать бизнес-метрики.
