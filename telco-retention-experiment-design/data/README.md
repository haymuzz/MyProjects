# Данные проекта

В проекте используется датасет **Telco Customer Churn** с Kaggle.

Ссылка на датасет: https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Файлы данных не добавлены в репозиторий, так как датасеты Kaggle могут быть крупными и требуют отдельного скачивания.

## Как скачать

```bash
kaggle datasets download -d blastchar/telco-customer-churn
unzip *.zip -d data/raw
```

После скачивания файлы нужно положить в папку:

```text
telco-retention-experiment-design/data/raw/
```
