# Данные проекта

В проекте используется датасет **Online Retail Customer Clustering** с Kaggle.

Ссылка на датасет: https://www.kaggle.com/datasets/hellbuoy/online-retail-customer-clustering

Файлы данных не добавлены в репозиторий, так как датасеты Kaggle могут быть крупными и требуют отдельного скачивания.

## Как скачать

```bash
kaggle datasets download -d hellbuoy/online-retail-customer-clustering
unzip *.zip -d data/raw
```

После скачивания файлы нужно положить в папку:

```text
customer-lifetime-value-segmentation/data/raw/
```
