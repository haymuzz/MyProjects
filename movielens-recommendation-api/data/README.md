# Данные проекта

В проекте используется датасет **MovieLens 20M Dataset** с Kaggle.

Ссылка на датасет: https://www.kaggle.com/datasets/grouplens/movielens-20m-dataset

Файлы данных не добавлены в репозиторий, так как датасеты Kaggle могут быть крупными и требуют отдельного скачивания.

## Как скачать

```bash
kaggle datasets download -d grouplens/movielens-20m-dataset
unzip *.zip -d data/raw
```

После скачивания файлы нужно положить в папку:

```text
movielens-recommendation-api/data/raw/
```
