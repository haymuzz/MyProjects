# Olist Marketplace Product Analytics

## Описание проекта

Продуктовая аналитика маркетплейса: заказы, доставка, выручка, отзывы и качество сервиса.

## Бизнес-задача

Маркетплейсу нужно понять, какие факторы сильнее всего влияют на выручку, скорость доставки и оценки клиентов.

## Датасет

- Kaggle: [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- Данные не хранятся в репозитории из-за размера и лицензии Kaggle.
- После скачивания положите файлы в `data/raw/`.

## Что будет сделано

- собрать витрину заказов из нескольких таблиц
- рассчитать GMV, AOV, order count, repeat rate
- проанализировать delivery SLA и просрочки
- найти категории и регионы с максимальной выручкой
- оценить связь между сроком доставки и review score
- подготовить рекомендации для продавцов и операционной команды

## Метрики

- GMV
- AOV
- repeat rate
- delivery delay
- review score
- orders by region

## Стек

SQL, Python, pandas, PostgreSQL-style queries, matplotlib, seaborn
