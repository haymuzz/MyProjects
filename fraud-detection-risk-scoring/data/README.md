# Данные проекта

В проекте используется датасет **Credit Card Fraud Detection** с Kaggle.

Ссылка на датасет: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Файлы данных не добавлены в репозиторий, так как датасеты Kaggle могут быть крупными и требуют отдельного скачивания.

## Как скачать

```bash
kaggle datasets download -d mlg-ulb/creditcardfraud
unzip *.zip -d data/raw
```

После скачивания файлы нужно положить в папку:

```text
fraud-detection-risk-scoring/data/raw/
```
