# Данные проекта

В проекте используется датасет **Brazilian E-Commerce Public Dataset by Olist** с Kaggle.

Ссылка на датасет: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Файлы данных не добавлены в репозиторий, так как датасеты Kaggle могут быть крупными и требуют отдельного скачивания.

## Как скачать

```bash
kaggle datasets download -d olistbr/brazilian-ecommerce
unzip *.zip -d data/raw
```

После скачивания файлы нужно положить в папку:

```text
olist-marketplace-product-analytics/data/raw/
```
